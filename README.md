# zerothon-DDT

#깃 사용방법
1. 자신이 할일을 이슈로 만든다.
     - 단위가 작을 수록 좋음(알아서 판단해서 편한대로 ㄱㄱ)
2. 이슈번호를 이용해서 브랜치를 만든다.
     - 브랜치 만드는 법
         - <>Code에서 branchs를 클릭한다.
         - New branch 클릭
         - 브랜치 이름은 [컨벤션]#[이슈번호]로 지정
         - 브랜치 생성
3. 로컬에서 같은 방법으로 브랜치 생성
4. 로컬에서 작업 후 새로만든 브랜치에 커밋
5. 작업 완료 후 로컬브랜치를 push
    - 새로 만든 브랜치에 추가
6. 문제없는 거 확인 후 풀리퀘로 main 브랜치에 추가
